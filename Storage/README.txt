Добро пожаловать в Comandrix!

Comandrix - суперпростой командный центр с:

 ---- менеджментом файлов
 ---- быстрой работы с терминалом
 ---- гибкой настройкой центра

Элементы управления:

 ---- кнопка '[(S)]' в левом нижнем углу отвечает за вызов панели управления
 ---- нажатие правой кнопки мыши по рабочему столу, вызывает контесктное меню с возможностью создания файла
